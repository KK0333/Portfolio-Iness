# Portfolio Website – Iness El Moujahdi

This repository contains the source code of a personal portfolio website developed as part of a university project.

The website presents my profile, skills, academic projects and contact information.

The project is built using **HTML, CSS and JavaScript**, with some layout utilities from **Bootstrap**.

# Project Structure

The project is composed of the following main files:

index.html  
Main HTML file containing the structure of the website.

style.css  
Contains all the styles for the website (layout, colors, typography, responsive design).

script.js  
Contains the JavaScript code used to add interactivity to the website.

# How to Run the Website
To run the project locally:

1. Download or clone the repository
2. Open the folder
3. Open the file **index.html** in a web browser

No additional installation or dependencies are required.

# Website Sections
The website contains several sections:

Hero section  
Introduction and navigation menu.

About  
Short presentation and academic background.

Skills  
Technical skills and tools used during my studies.

Projects  
Examples of academic projects completed during the bachelor's degree.

Languages  
Languages spoken.

Contact  
Email and LinkedIn information.

# JavaScript Functionality

The JavaScript file adds some interactivity:

Scroll-to-top button  
A button appears when the user scrolls down the page and allows them to return to the top smoothly.

Section animations  
Sections appear progressively when scrolling.

Title hover effect  
A small animation effect on the main title.

# Responsive Design
The website is responsive and adapts to different screen sizes using **CSS media queries**.

Layouts are adjusted for:
- desktop screens
- tablets
- mobile devices
